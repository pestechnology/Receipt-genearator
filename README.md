\# Receipt Generator (Python)



A beginner-friendly console-based Python project to generate food receipts.



\## What It Does

\- Displays a food menu

\- Takes item selection and quantity

\- Calculates subtotal, tax, and final amount

\- Generates a receipt

\- Saves and displays previous bills

\- Includes basic automated tests using pytest



\## How to Run

```bash

python main.py



How to Run Tests:

python -m pip install -r requirements.txt

python -m pytest
